# WEEKLY_ASSESSMENT_2
This project leverages Pandas to analyze a comprehensive dataset of Indian startup funding rounds. The goal is to showcase how Python’s data science stack can transform raw CSV data into actionable insights about industries, investors, and funding trends.

Key features include:

Data Cleaning: Handling inconsistent city names, funding stages, and currency formats.

Exploratory Analysis: Summarizing funding amounts by industry, city, and stage.

Investor Insights: Identifying top investors and their recurring patterns across startups.

Time-Series Trends: Analyzing funding growth over time to spot sectoral booms (e.g., FoodTech, AI/ML).

Visualization: Generating bar charts and trend lines to make insights shareable and presentation‑ready.

Case Studies: Highlighting startups like Ola, Swiggy, and Meesho to demonstrate funding evolution.

💡 Built entirely with Python fundamentals and Pandas, this repository is a practical example of how data wrangling and analysis can uncover meaningful stories in startup ecosystems.

"Decode India’s startup funding landscape — from raw CSVs to clear insights."
